# Adobe Experience Manager의 무의미한 진화

Adobe Experience Manager이 클라우드 기반의 차원을 넘어 실제로 아젠틱한 차원으로 진화하는 방법을 알아보십시오. 콘텐츠 워크플로를 자동화하고 개발자 생산성을 향상시키는 AEM 에이전트, Content AI 기본 서비스 및 API 통합에 대해 알아봅니다. 업계 전문가인 세드릭 휴슬러(Cedric Huesler)와 마틴 부어지(Martin Buergi)가 새너제이(San Jose)에서 실시간으로 기록한 혁신적인 기술을 공개합니다.

>[!VIDEO](https://video.tv.adobe.com/v/3477593/?learn=on&enablevpops)

Adobe Developers Live 2025를 지원해준 우리의 후원자인 Algolia와 Ensemble에게 특별한 감사를 드립니다.

이 세션(Adobe Experience Manager의 아젠틱한 진화)에서는 AEM이 어떻게 클라우드 기반의 것을 넘어 아젠틱하게 진화하고 있는지 자세히 살펴봅니다. 콘텐츠 워크플로를 자동화하고 개발자 생산성을 향상시키는 AEM 에이전트, Content AI Foundation Services 및 API 통합 기능을 공개하는 Cedric Huesler 및 Martin Buergi와 함께 하십시오. San Jose에서 생방송으로 녹화됨.
