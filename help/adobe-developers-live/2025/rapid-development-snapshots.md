# 동결, 복원, 혁신: 클라우드 개발을 위한 신속한 개발 환경 스냅샷으로 실습

Adobe Experience Manager Cloud Service의 RDE(신속한 개발 환경)가 어떻게 신속한 클라우드 기반 반복을 가능하게 하는지 알아보십시오. 스냅샷 기능을 통해 개발자는 전체 RDE 상태를 동결, 복원 및 재사용할 수 있습니다. 이를 통해 Adobe I/O CLI RDE 플러그인을 사용하여 테스트를 간소화하고, 즉시 롤백하고, 개발 주기를 가속화할 수 있습니다. 클라우드 개발 효율성을 높이고 기업 거버넌스 정책에 부합합니다.

>[!VIDEO](https://video.tv.adobe.com/v/3477705/?learn=on&enablevpops)

이 세션 — 고정, 복원, 혁신: 클라우드 개발을 위한 신속한 개발 환경 스냅샷으로 실습 — Adobe의 소프트웨어 개발 엔지니어인 Remo Liechti가 참여했으며 Adobe Developers Live 2025용으로 사전 녹음되었습니다. Remo는 빠른 개발 환경(RDE)이 Adobe Experience Manager Cloud Service에서 빠른 클라우드 기반 반복을 가능하게 하는 방법과 새로운 스냅샷 기능을 통해 개발자가 전체 RDE 상태를 동결, 복원 및 재사용할 수 있는 방법을 보여 줍니다. Adobe I/O CLI RDE 플러그인을 사용하여 AEM Cloud에서 테스트를 간소화하고, 즉시 롤백하고, 개발 주기를 가속화하는 방법에 대해 알아봅니다.

Adobe Developers Live 2025를 지원해준 우리의 후원자인 Algolia와 Ensemble에게 특별한 감사를 드립니다.
