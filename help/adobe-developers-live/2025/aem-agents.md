# Adobe Experience Manager 에이전트 소개

Adobe Developers Live 2025: 산호세, 시드니, 방갈로르, 바젤 지역의 개발자, 설계자, 기술자들이 모여 Adobe Experience Manager과 Commerce을 통해 AI와 Edge Delivery Services이 디지털 경험을 어떻게 재정의하는지 살펴보는 Agentic Web 구축

산호세에서 실시간으로 녹화된 이 세션에서는 웹 사이트 마이그레이션을 혁신하는 AI 기반 기능인 Adobe Experience Manager Agents를 소개합니다. 원활한 GitHub 통합을 통해 AEM 에이전트는 블록 매핑, 콘텐츠 가져오기 및 스타일을 깔끔한 모범 사례 코드로 자동화하여 Edge Delivery Services에 온보딩을 가속화합니다. 개발자가 마이그레이션을 단순화하고 게재 시간을 단축하며 검색 및 생성 엔진에 최적화된 사이트를 시작하는 방법에 대해 알아봅니다.
