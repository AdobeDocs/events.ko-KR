# EDS(Edge Delivery Services)를 사용한 Qantas.com 현대화

Qantas가 Adobe Experience Manager Edge Delivery Services을 사용하여 웹 사이트를 개편한 방법을 살펴보십시오. 이 변환은 게시 시간을 몇 주에서 몇 분으로 단축하고, 개발을 간소화했으며, 의미 마크업을 간소화하여 SEO를 개선했습니다. 온프레미스 배포에서 에지 아키텍처로의 전환과 JavaScript 기반 워크플로우로 인재 풀을 확장하는 방법에 대해 알아봅니다. 이 최신 접근 방식을 통해 작성 속도를 높이고 기업 내부 규정 목표를 지원하는 방법에 대한 통찰력을 얻으십시오. 대화에 참여하고 다가오는 Adobe 이벤트를 탐색하여 보다 깊이 있는 통찰력을 얻으십시오.

>[!VIDEO](https://video.tv.adobe.com/v/3477681/?learn=on&enablevpops)

이 Fireside 채팅 - Edge Delivery Services을 통한 현대화 Qantas.com - Accenture Song의 솔루션 설계자인 Tim Lavelle와 Adobe의 수석 제품 전문가인 Glen Lawson이 시드니에서 실시간으로 녹음을 했습니다. 이들은 Qantas의 주요 재구축 프로젝트와 Adobe Experience Manager Edge Delivery Services으로의 전환에 대해 논의하며, 온프레미스 배포에서 에지 아키텍처로 전환하면 게시 시간이 몇 주에서 몇 분으로 단축되는 방식을 중점적으로 다룹니다. 이러한 현대적인 접근 방식이 개발을 간소화하고, JavaScript 기반 워크플로우를 통해 인재 풀을 확장했으며, 더 깔끔한 시맨틱 마크업과 유니버설 편집기를 통해 작성 속도와 SEO를 개선하는 방법을 알아봅니다.

Adobe Developers Live 2025를 지원해준 우리의 후원자인 Algolia와 Ensemble에게 특별한 감사를 드립니다.
