# 엔진 내부: Adobe Experience Manager Sites Optimizer을 사용한 코드 최적화

Adobe Experience Manager Sites Optimizer이 중복 스크립트 및 무거운 DOM 구조를 탐지하여 개발자가 코드 품질을 개선하는 데 어떻게 도움이 되는지 알아보십시오. 이 세션에서는 이 도구를 사용하여 핵심 웹 바이탈을 개선하고 기술적 부채를 줄이며 성능을 향상시키는 방법을 보여줍니다. 워크플로우 내에서 권장 사항을 원활하게 적용하여 보다 빠르게 수정하고, 보다 강력한 SEO를 구현하고, 보다 원활한 사용자 경험을 제공하는 방법을 알아봅니다.

>[!VIDEO](https://video.tv.adobe.com/v/3477724/?captions=kor&learn=on&enablevpops)

이 세션(엔진 내: Adobe Experience Manager Sites Optimizer을 통한 코드 최적화)은 Deck Reyes가 Sites Optimizer이 프런트 엔드 및 백 엔드 구현을 분석하여 중복 스크립트, 무거운 DOM 구조 및 핵심 웹 Vitals 병목 현상을 탐지하는 방법을 보여 주는 기능입니다. AEM에서 직접 기술적인 문제를 줄이고 성능을 개선하기 위해 권장 사항을 적용하는 방법을 알아봅니다. San Jose에서 생방송으로 녹화됨.

Adobe Developers Live 2025를 지원해준 우리의 후원자인 Algolia와 Ensemble에게 특별한 감사를 드립니다.
