# AEM as a Cloud Service 2024.5.0 릴리스 업데이트

이번 릴리스의 기능은 다음에 중점을 둡니다.

* AEM 및 Dynamic Media용 새로운 게시 옵션
* 유니버설 편집기 개선 사항
* 적응형 Forms Foundation 구성 요소를 핵심 구성 요소로 마이그레이션
* 적응형 Forms에서 추가 CAPTCHA 지원
* Cloud Foundation 개선 사항
