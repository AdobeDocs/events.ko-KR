# AEM as a Cloud Service 2023.6.0 릴리스 업데이트


이 릴리스의 기능은 Adobe Experience Manager [!DNL Assets] 혁신에 중점을 두며 다음을 포함합니다.

* DAM용 새 사용자 인터페이스
* 검색 개선 사항
* 3D 자산 관리
* 메타데이터 관리 개선 사항
* AEM [!DNL Assets] + [!DNL Frame io] 통합
