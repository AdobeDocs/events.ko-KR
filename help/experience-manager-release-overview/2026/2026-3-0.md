# 2026.03.0 AEM 릴리스 개요 비디오

>[!VIDEO](https://video.tv.adobe.com/v/3483060/?learn=on&enablevpops)

이 릴리스의 기능은 Experience Manager Sites, Forms, Assets 및 Cloud Foundation의 혁신에 중점을 두며, 다음과 같은 사항을 포함합니다.

* 사이트
   * Experience Production Agent 업데이트 및 시작 지원

* 양식
   * IC 편집기에 대한 워크플로우 지원
   * 핵심 구성 요소에 대한 테마 편집기 지원
   * Forms에 대한 버전 관리 지원

* 자산
   * AEM Content Advisor - GA(Adobe 앱 포함)
   * Microsoft Office 커넥터(PowerPoint, Word, Excel)

* 클라우드 기반
   * 거버넌스 에이전트: 브랜드 관리
   * Touch UI에 대한 권한 감사
   * OIDC 로그아웃 기능
   * 유지 보수 업데이트 일시 중지

* Cloud Manager
   * Cloud Manager 서버
