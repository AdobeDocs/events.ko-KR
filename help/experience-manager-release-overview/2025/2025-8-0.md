# 2025.08.0 릴리스 개요 비디오

## 클라우드 기반

* AEM CMK에 안전한 연결 및 자동화를 제공하여 고객의 보안 및 신뢰를 강화합니다&#x200B;.
* Productise OIDC for publish - Jira
* Bring Your Own Git (BYOG)

## 양식

* 보다 쉬운 규칙 생성을 통한 작성자 효율성
* DoR을 다운로드하는 OOTB 버튼

* 마크업 가져 오기
* 콘텐츠 보안 정책
