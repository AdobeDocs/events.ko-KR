# Workfront Fusion + JSONata를 사용하여 JSON 활용

Steve Watts와 Adobe Professional Services의 Brian Haight와 함께 호스팅된 이 세션은 JSON 경로 표기법을 읽고 데이터 유형을 이해하고 Fusion에서 올바른 시작 경로를 선택하는 기본 사항을 안내했습니다.

또한 데이터 찾기 및 필터링부터 JSON 재구성, 즉석으로 고급 변환을 수행하는 것까지 세 가지 실제 사례를 다룹니다. 시간을 절약하고 데이터 처리를 단순화하고자 하는 모든 Fusion 관리자에게 꼭 필요한 기술입니다!

>[!VIDEO](https://video.tv.adobe.com/v/3477129/?learn=on&enablevpops)

라이브로 참석할 수 없는 경우 걱정하지 마십시오. [슬라이드 데크](https://workfront-experience.s3.us-west-2.amazonaws.com/Training/Guides/Customer+Success+at+Scale/Slide+Deck+-+JSONata+Fusion+Event+-+September+17%2C+2025.pdf)를 검토하고 필요할 때 녹화를 시청하면 됩니다.
