# 새로운 Adobe Campaign Web User Interface의 발전

Adobe Campaign v8은 완전히 새로운 모양과 느낌을 선사할 것입니다. 다시 상상한 사용자 경험은 캠페인 디자인 프로세스를 단순화하고 다른 Adobe Experience Cloud 솔루션과 일치합니다.
