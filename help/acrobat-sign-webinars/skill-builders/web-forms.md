# Acrobat Sign에서 웹 양식 시작하기

>[!VIDEO](https://video.tv.adobe.com/v/3491645/?learn=on&enablevpops)

여전히 이메일을 주고 받거나 한 번에 하나의 계약서를 추적합니까? Acrobat Sign의 Web Forms를 사용하면 아무것도 보내지 않고 웹 사이트, 공유 링크 또는 포함된 페이지에서 누구나 액세스하고, 채우고, 서명할 수 있는 단일 서명 양식을 게시할 수 있습니다. 전문가들이 다루는 실습형 웨비나를 시청하십시오.

* 웹 양식은 무엇이며 어디에서 빛을 발합니까 - 수백 명의 직원이 완료할 수 있는 신규 고용 온보딩 패킷을 게시하는 HR 팀부터 대규모 서명을 수집하기 위해 웹 사이트에 고객 대면 애플리케이션을 게시하는 재무 팀까지!
* 웹 양식을 만들고 시작하는 방법 - 문서나 템플릿 업로드, 양식 필드 배치, 서명자 인증 구성 및 공유 가능한 URL 또는 포함 가능한 코드로 양식 게시를 포함하여 첫 번째 웹 양식을 만드는 방법에 대해 실시간 연습합니다.
* 웹 양식 결과 관리 및 추적 방법 - 제출을 실시간으로 모니터링하고, 양식 필드 데이터를 다운로드하고, 라이브 양식을 편집 또는 비활성화하고, 사용량이 증가함에 따라 웹 양식을 체계적으로 구성합니다.

10명 또는 1만명의 서명을 수집하는 경우에도 안심하고 첫 번째 웹 양식을 작성, 게시 및 관리할 수 있도록 이 세션을 남겨둡니다.
